## 产品介绍

店赢OS是新一代AI驱动的智能门店运营系统，基于Multi-Agent架构，让实体门店实现"一人公司"的运营模式。

### 核心价值

- **一个人运营无限门店**：多Agent协同，AI虚拟店长7×24小时自主决策
- **差评自动处理**：智能识别、自动回复、持续追踪，差评变转机
- **动态定价**：天气、时段、竞品、节假日综合考量，自动调价
- **跨店知识迁移**：一家店的成功经验，复制到N家店

1. **差评处理** - 差评预警 → 原因分析 → 方案生成 → 自动回复
2. **动态定价** - 天气分析 → 策略建议 → 自动同步 → 效果追踪
3. **VIP管理** - 客户分析 → 流失预警 → 个性化召回 → 持续运营
4. **运营建议** - 数据诊断 → 问题定位 → 优化方案 → 执行落地
5. **跨店迁移** - 方案分析 → 可行性评估 → 适配调整 → A/B测试
6. **数字孪生** - 多方案模拟 → 条件对比 → 最佳推荐 → 自动复盘

## 技术架构

### Multi-Agent系统

```
┌─────────────────────────────────────────────────────────────┐
│                        用户输入                               │
│              (老板提问 / 差评预警 / 运营决策)                  │
└─────────────────────────────────────────────────────────────┘
                            ↓ 调度
┌─────────────────────────────────────────────────────────────┐
│                     AI虚拟店长 (核心调度)                      │
│              统一调度各子Agent，负责任务分配与协调               │
└─────────────────────────────────────────────────────────────┘
                            ↓ 反馈
┌─────────────────────────────────────────────────────────────┐
│  线索Agent   │   方案Agent   │   交付Agent   │   运营Agent    │
│   分析根因    │   生成策略    │   执行操作     │   持续优化     │
└─────────────────────────────────────────────────────────────┘
                            ↓ 沉淀
┌─────────────────────────────────────────────────────────────┐
│                    10个行业知识库                              │
│  餐饮/零售/休闲娱乐/健身美容/教育培训/生活服务...              │
└─────────────────────────────────────────────────────────────┘
```

### 五级信任体系

| 等级 | 说明 | 权限范围 |
|------|------|----------|
| A级 | 优秀 | 全功能开放，AI自主决策 |
| B级 | 良好 | 全功能，重要操作需确认 |
| C级 | 标准 | 核心功能，操作需审核 |
| D级 | 观察 | 基础功能，观察学习期 |
| E级 | 预警 | 最小权限，需人工干预 |

## 快速开始

### 在线体验

直接访问：[https://liuhuanxi-oss.github.io/dianying-os/](https://liuhuanxi-oss.github.io/dianying-os/)

### 本地运行

1. 克隆项目
```bash
git clone https://github.com/liuhuanxi-oss/dianying-os.git
cd dianying-os
```

2. 使用任意HTTP服务器启动
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```

3. 浏览器打开 `http://localhost:8000`
