# 店赢OS 商户管理系统 MVP

## 🚀 快速开始

1. **打开登录页**：直接在浏览器中打开 `login.html`
2. **快速体验**：点击"快速体验演示版"按钮直接进入系统
3. **完整登录**：输入手机号（任意11位数字），点击获取验证码后自动填入"123456"进行登录

## 📱 4大核心功能

### 1. 🤖 差评全自动处理
- **位置**：左侧菜单 → "差评处理"
- **功能**：
  - 实时显示待处理差评列表
  - AI智能分析差评类型、情绪、影响程度
  - 一键生成3种不同风格的回复（温和安抚型/补偿方案型/正面解释型）
  - 一键同步到所有平台
- **演示数据**：3条待处理差评（美团、饿了么、抖音）

### 2. 💹 竞品智能定价
- **位置**：左侧菜单 → "智能定价"
- **功能**：
  - 监控竞品价格变动
  - AI给出调价建议（基于竞品价格、成本、销量等）
  - 一键同步调价到全平台
- **演示数据**：2条价格变动提醒（1条降价需跟进、1条涨价可跟进）

### 3. 📱 8平台统一管理
- **位置**：左侧菜单 → "多平台管理"
- **功能**：
  - 查看8大平台绑定状态（美团/饿了么/抖音/大众点评/高德/口碑/微信/小红书）
  - 统一数据看板（总营收、总订单、评分）
  - 各平台营收对比图表
  - 批量操作（统一回复/发布活动/上下架/同步库存）
- **演示数据**：6个平台已连接，2个待绑定

### 4. 📊 自动对账系统
- **位置**：左侧菜单 → "自动对账"
- **功能**：
  - 自动拉取8平台账单
  - 智能识别异常订单
  - 统一对账单展示
  - 一键导出Excel/PDF报表
- **演示数据**：对账成功率98.5%，3条异常订单
