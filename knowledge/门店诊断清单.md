# 门店诊断清单

> 适用维度：服务 / 出品 / 环境 / 营销 / 财务  
> 使用方式：每周自检一次，每月深度检一次，发现问题立即整改  
> 参考来源：中国连锁经营协会《门店运营督导手册2025》

---

## 一、服务维度（总分100分，合格线75分）

### 1.1 基础服务检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 员工是否统一着装、佩戴工牌 | 干净整洁、无污渍 | |
| 员工是否在岗前完成仪容仪表自查 | 对镜检查、微笑练习 | |
| 员工是否使用标准迎宾话术 | "您好，欢迎光临" | |
| 员工是否主动介绍菜品/活动 | 不等客人问就开口 | |
| 员工是否在3秒内响应客人需求 | 目光接触、点头示意也算 | |

### 1.2 响应速度检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 客人入座后多久有人倒水 | 1分钟内 | |
| 点单后多久确认 | 3分钟内 | |
| 上菜速度是否达标 | 快餐<15分钟，正餐<25分钟 | |
| 结账等待时间 | <3分钟 | |
| 投诉响应时间 | 店长5分钟内到场 | |

### 1.3 服务细节检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 是否有员工手机响铃/玩手机 | 不允许 | |
| 员工之间是否交流工作内容 | 不允许闲聊 | |
| 客人离店是否有送客话术 | "欢迎下次光临" | |
| 员工是否记住老客姓名/口味 | 熟客进门能叫出名字 | |

## 二、出品维度（总分100分，合格线80分）

### 2.1 食品安全检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 食材是否在保质期内 | 先进先出原则 | |
| 冷藏食材温度是否达标 | ≤4℃ | |
| 冷冻食材温度是否达标 | ≤-18℃ | |
| 生熟分开存放 | 无交叉污染 | |
| 制作过程是否戴口罩/手套 | 接触食品必须 | |

### 2.2 品质标准检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 每日备菜前是否试味 | 确保标准统一 | |
| 菜品份量是否符合标准 | 误差<10% | |
| 摆盘是否美观 | 与标准图对比 | |
| 温度是否达标 | 热菜>60℃，凉菜<4℃ | |
| 异物检查 | 出餐前必须检查 | |

### 2.3 设备状态检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 灶具是否正常工作 | 火候稳定 | |
| 冷藏冷冻设备是否正常 | 温度计确认 | |
| 保温设备是否到位 | 必要时使用 | |
| 消毒设备是否正常 | 消毒柜灯亮 | |

## 三、环境维度（总分100分，合格线70分）

### 3.1 视觉环境检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 门头灯是否正常 | 晚上必须亮 | |
| 海报/POP是否过期 | 活动结束后48小时内撤除 | |
| 桌椅是否干净无损坏 | 破损立即维修或更换 | |
| 地面是否干净无污渍 | 随时清洁 | |
| 灯光亮度是否合适 | 不刺眼、不昏暗 | |

### 3.2 卫生环境检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 桌面是否消毒清洁 | 每餐后必须擦 | |
| 卫生间是否干净无异味 | 每2小时检查一次 | |
| 垃圾桶是否及时清理 | 不得超过2/3满 | |
| 厨房是否整洁有序 | 工具归位 | |
| 空调/通风是否正常 | 无异味 | |

### 3.3 安全环境检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 消防通道是否畅通 | 无杂物 | |
| 监控是否正常工作 | 无死角 | |
| 应急灯是否正常 | 测试一下 | |
| 电源线是否杂乱 | 必须整理 | |
| 防滑措施是否到位 | 湿拖地后放警示牌 | |

## 四、营销维度（总分100分，合格线65分）

### 4.1 线上运营检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 大众点评评分是否≥4.3 | 每日查看 | |
| 差评是否24小时内回复 | 必须及时 | |
| 菜单/价格是否与平台一致 | 防止顾客到店失望 | |
| 团购套餐是否正常售卖 | 库存充足 | |
| 是否有新上评价/笔记 | 每周至少1条 | |

### 4.2 会员运营检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 会员日活动是否执行 | 固定日期必须做 | |
| 沉睡会员是否被唤醒 | 30天未消费发券 | |
| 新客是否被引导注册会员 | 结账时必问 | |
| 会员生日礼是否发送 | 提前3天提醒 | |
| 会员积分是否正常累计 | 查账核对 | |

### 4.3 活动执行检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 本周活动是否按计划执行 | 有方案必执行 | |
| 活动物料是否到位 | 海报、道具、赠品 | |
| 员工是否知晓活动内容 | 晨会培训过 | |
| 活动数据是否记录 | 参与人数、转化率 | |
| 活动效果是否复盘 | 结束后24小时内 | |

## 五、财务维度（总分100分，合格线70分）

### 5.1 营收检查

| 检查项 | 检查标准 | 是/否 |
|--------|----------|-------|
| 当日营收是否达标 | 对比昨日/上周同期 | |

| 退款/免单是否有记录 | 任何减免必须登记 | |
| 日结是否按时完成 | 当日24点前 | |

| 水电费是否异常 | 对比上月同期 | |
| 耗材使用是否合理 | 一次性用品控制 | |
| 库存周转是否正常 | 避免积压 | |
